# 6. PolyFills

Los polyFills sirven para que las funcionalidades mas modernas como los webcomponent funcione en navegadores mas antiguos. Independientemente de su version etc. Los polyfills no siempre funcionan al 100%.

En la pagina __www.webcomponents.org__ explica los polyfills que hay y como usarlos. Ademas que cosas pueden dar problemas. Como por ejemplo la etiqueta :host de css.

https://www.webcomponents.org/polyfills
